Что реализовано

  • REST API Контактная форма, health check и metrics endpoint.
  • AI fallback Если AI недоступен, сервис продолжает обработку заявки.
  • Rate limiting Защита от спама по IP через файловое хранение.
  • Логи и метрики Запросы, ошибки и статистика сохраняются локально.
